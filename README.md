# 🚀 Leadership Dashboard Project

A modern **React-based dashboard** powered by an **Excel data source** to visualize and evaluate leadership performance across departments.

The frontend is built using **React**, while data is dynamically fetched from a `leadership.xlsx` file (stored in the public folder) using the **Sheet.js** library.

---

## 📊 Overview

This dashboard is designed to help **Colombo Dockyard** analyze and rank departments based on their:

* 🧠 Leadership qualities
* 🤝 Team collaboration
* 📁 Project performance

Each department participates in assigned projects and is evaluated based on their execution and contribution, resulting in a clear ranking system.

---

## ✨ Key Features

* 🏆 **Department Rankings**
  Displays rankings based on overall project performance and leadership evaluation

* 📁 **Project Insights**
  View project details including descriptions and assigned departments

* 📈 **Participation Metrics**
  Track key statistics such as:

  * ✅ Project completion percentage
  * 👥 Number of participants
  * 📊 Allocation vs participation ratio

---

## ⚙️ How It Works

* 📂 Data is stored in `leadership.xlsx` inside the public folder
* 🔍 **Sheet.js** reads and extracts data from specific rows and columns
* ⚛️ React dynamically renders the processed data into a user-friendly dashboard

---

## 🛠️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/LeadershipProject.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd LeadershipProject
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Application

```bash
npm start
```

---

## 📂 Important Notes

* Ensure the `leadership.xlsx` file is placed inside the **public folder**
* Modify the Excel file carefully — column names and structure should remain consistent for accurate data fetching
* The app reads data based on **column names and row positions**
