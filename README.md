# LeadershipProject
A React-based dashboard powered by an Excel data source. The frontend is built using React, while data is dynamically fetched from a leadership.xlsx file (stored in the public folder) using the Sheet.js library.

The dashboard visualizes department performance within Colombo Dockyard, ranking teams based on their project execution, teamwork, and leadership outcomes.

Key Features:

Displays department rankings based on project performance and leadership evaluation

Shows project details, including descriptions and assigned departments

Tracks participation metrics such as completion percentage, participant count, and allocation ratios
